# Phi 2

[:hugs: Microsoft / `phi-2`](https://huggingface.co/microsoft/phi-2#sample-code)

## Setup

```bash
# One time.
conda env create -f conda.yaml

conda activate phi2

# If changing conda.yaml
conda env update -f conda.yaml
```
